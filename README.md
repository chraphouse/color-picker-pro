# Color Picker Pro

A Chrome extension for picking colors from any webpage using the EyeDropper API. Supports HEX, RGB, and HSL formats with a 24-swatch color history.

## Features

- Pick colors from any webpage using the EyeDropper API
- Output in HEX, RGB, and HSL formats
- Color history storing up to 24 swatches
- Manual color input field
- One-click copy to clipboard

## Keyboard Shortcuts

- **Alt+C** - Activate the color picker

## Installation

1. Download or clone this repository
2. Open `chrome://extensions/` in Chrome
3. Enable **Developer mode** (top right toggle)
4. Click **Load unpacked** and select this directory

## Built with
